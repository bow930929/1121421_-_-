# 1121421_許志文_房貸計算器

## 專案簡介
使用者輸入房屋總價、自備款比例、貸款利率、貸款年限及寬限期後，
程式將自動計算每月應繳金額、首期利息與本金、總利息支出及總還款金額。

## 使用說明
1. 輸入房屋總價（元）
2. 輸入自備款比例（%），預設 20%
3. 輸入貸款利率（年利率 %），預設 2.15%
4. 輸入貸款年限（年），預設 20 年
5. 寬限期為選填，若不填預設為 0
6. 按下「計算」即顯示試算結果
7. 按下「清除」可重置所有欄位
